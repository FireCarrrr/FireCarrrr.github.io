SpringBoot 启动流程

SpringApplication 类里面封装了整个 SpringBoot 引用启动的流程。SpringApplication#run() 也是整个 SpringBoot 应用的入口。

SpringApplication

SpringApplication 的 Java doc ：

Class that can be used to bootstrap and launch a Spring application from a Java main method. By default class will perform the following steps to bootstrap your application:

  • Create an appropriate ApplicationContext instance (depending on your classpath)
  • Register a CommandLinePropertySource to expose command line arguments as Spring properties
  • Refresh the application context, loading all singleton beans
  • Trigger any CommandLineRunner beans

SpringApplication 是用来从 Java main 方法引导和启动一个 Spring 应用程序的类。默认情况下，这个类会以下面的步骤引导引用程序：

  • 创建一个适当的 ApplicationContext 实例（基于你的 classpath）
  • 注册一个 CommandLinePropertySource，将命令行参数暴露为 Spring properties
  • 启动 application context，加载所有单例的 bean
  • 触发任意一个 CommandLineRunner bean
